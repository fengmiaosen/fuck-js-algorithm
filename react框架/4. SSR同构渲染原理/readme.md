
## Next.js

next.js功能

路由： 基于“pages”的静态/动态路由
数据获取： 针对不同场景的数据获取API
css支持： 全局样式、组件级样式、CSS-in-JS
TypeScript： 集成类似IDE的TS体验，开箱即用
环境变量： 内置对于环境变量的支持，并能导出至浏览器环境
静态文件服务： 提供图片、文本、甚至html等静态文件支持
