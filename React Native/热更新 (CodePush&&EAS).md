
## 热更新 (CodePush)

https://github.com/microsoft/code-push-server?tab=readme-ov-file

CodePush 是 App Center 的一项功能，使 React Native 开发者能够直接向用户设备部署移动应用更新。
它由两部分组成：

1. [CodePush 服务器](https://github.com/microsoft/code-push-server/blob/main/api/README.md)，开发者可以在此发布应用更新（如 JS、HTML、CSS 或图像变更）
2. (CodePush React Native 客户端 SDK)(https://github.com/Microsoft/react-native-code-push)，使应用能够查询更新。
