About TTRAA
关于 TTRAA

TTRAA is the operating system for modern advertising in MENA—one platform to discover, book, and measure campaigns across Influencers, TV, OOH, premium Display, and Podcasts. Modules: TTRAA Discovery, TTRAA Campaigns, and TTRAA Intelligence (our integrated social listening & analytics), built API-first.
TTRAA 是面向中东和北非（MENA）地区的现代广告“操作系统”——在一个平台上完成广告发现、预订与效果衡量，覆盖网红、电视、户外（OOH）、高端展示广告以及播客等渠道。核心模块包括：TTRAA Discovery、TTRAA Campaigns 和 TTRAA Intelligence（我们整合的社媒监听与分析），以 API 优先架构构建。

Role Summary
职位总结

Own the platform—SRE, Architecture, AI & Data Pipelines, API Design, DevInfra—to deliver a secure, reliable, cloud-native foundation. Services run on Kubernetes, APIs delivered via Kong, stack is Node.js + TypeScript. Partner closely with the Director of Product Engineering on contracts/SLAs, performance, and releases. Help build the new tech hub team; we currently have a single tech hub in Cairo, Egypt with around 50 engineers, and our commercial team and headquarters are in Riyadh, Saudi Arabia.
全面负责平台：包括 SRE、架构、AI 与数据管道、API 设计、开发基础设施，交付安全、可靠、云原生的技术底座。服务运行在 Kubernetes 上，API 通过 Kong 暴露，技术栈为 Node.js + TypeScript。与产品工程总监紧密合作，围绕合同/SLAs、性能和发布开展工作。参与建设新的技术中心团队；目前我们在埃及开罗有一个技术中心，约 50 名工程师，商业团队与总部位于沙特阿拉伯利雅得。
